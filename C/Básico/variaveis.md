# Variáveis

1. Crie uma variável `a` de tipo `int`.

2. Crie uma variável `b` de tipo `int` inicializada com o valor `289`.

3. Observe o seguinte trecho de código.

   ```c
   int a = 10;
   {
       int a;

       a = 5;
   }
   ```

   Qual será o valor de `a` após a execução desse trecho?

## Respostas

1. `int a;`

2. `int b = 289;`

3. `10`
